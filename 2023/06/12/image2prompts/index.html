Stable Diffution图生文-Kaggle竞赛 'Stable Diffusion - Image to Prompts'Kaggle Competition

2023-06-12
2023-06-13
Machine Learning

该竞赛要求模型基于给定生成图片来推理对应文本指令（text prompts），并通过文本嵌入的余弦相似度衡量模型预测结果的表现。

数据处理：由于本次比赛没有给出训练集数据，且线上测试数据完全封闭，所以只能基于StableDiffusion模型自己构造训练集与验证集。首先利用现有的DiffusionDB数据集文本（过滤文本相似度高于0.8的数据），划分部分用于构建可靠的验证集。接着利用几个开源数据集以及文本语言预训练模型生成大量的文本指令（过滤文本相似度高于0.8的数据）。最后，将所有文本指令利用StableDiffusionV2.0模型逐一生成图片数据集。
训练阶段：使用OpenCLIP的预训练模型ConvNext-XXLarge，修改预处理器，冻结约3/4层数在大尺寸512x512训练一定轮次，得到稳定的推理模型。
推理阶段：使用同样的数据处理步骤在线上使用训练完成的模型预测最终的结果。

Data processing: Since no training set data is given in this competition and the online test data is completely closed, we can only construct our own training and validation sets based on the StableDiffusion model. Firstly, the existing DiffusionDB dataset text (filtering data with text similarity higher than 0.8) was used to divide the part for constructing a reliable validation set. Next, a large number of text instructions (filtering data with text similarity higher than 0.8) are generated using several open source datasets and a text language pre-training model. Finally, all text instructions are used to generate image datasets one by one using the StableDiffusionV2.0 model.
Training phase: Using the OpenCLIP pre-training model ConvNext-XXLarge, the pre-processor is modified to freeze about 3/4 layers in large size 512x512 to train a certain number of rounds to get a stable inference model.
Inference phase: Use the same data processing steps online to predict the final result using the completed training model.

DiffusionDB: https://www.kaggle.com/datasets/alexandreteles/diffusiondb-metadata
Conceptual Captions: https://huggingface.co/datasets/conceptual_captions
pszemraj data: https://huggingface.co/datasets/pszemraj/text2image-multi-prompt
LAION COCO: https://huggingface.co/datasets/laion/laion-coco
Prompts pre-trained model: https://huggingface.co/Gustavosta/MagicPrompt-Stable-Diffusion
